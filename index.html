What is Css box model

The term "box model" in Css is used when refering to design and layout.
Simply, It's a box that wraps around every HTML element.

The properties of a Css box model


    Padding:add the background to the element to make goes around the content.
    Margin:space out two element apart from each other.
    Border:It's a border/ block that goes around the padding and content.
    Content:refers to the inner section of the box where text and images appear
    Height :specify the distinct size of the content.
    Width: refers to how larger the border is.

Padding, margins, borders and content are essential properties of Css box model.

How to calculate the dimension of a box model

The total width of an element should be calculated like this:
Total element width = width + left padding + right padding + left border + right border + left margin + right margin.
Tatal dimension= 400px+20px+20p+2px=442px
